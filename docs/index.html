Features

One job, done quietly.

End-to-end encrypted

Only the two of you

QUIC + TLS 1.3 keyed by per-install ed25519 identities. Relays carry ciphertext only.

Iroh QUIC

Direct when it can, relayed when it can't

NAT hole-punching by default. Falls back to n0's public relays — no infrastructure to run.

Pair once

Share a code, then forget about it

Paste a friend's code the first time. After that, click their contact tile and drop a file.

Per-file accept

Nothing arrives unwelcomed

The receiver sees an Accept / Decline prompt for every offer with the filename and size.

Live presence

Green dot, simple state

Contacts show online / offline via a quiet background ping. Click to dial.

Tauri · Rust

Native Linux

Small binary, AppImage and .deb, in-app updater. No Electron, no daemon.

Principles

What this isn't.

No store-and-forward. If the receiver isn't online, the sender waits. We never hold anyone's files.
No accounts. Your identity is a keypair on your disk. Lose it, lose the link.
No krill servers. n0's public iroh relays do the introductions. We operate nothing.
No telemetry. The app makes one network call you didn't initiate: the in-app updater check.
Install

Two files. One minute.

Both formats ship the same binary. The Copy button in the Files view also needs wl-clipboard (Wayland) or xclip (X11) — almost everyone has one already.

Or build from source: pnpm install && pnpm tauri build.

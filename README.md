# File Drop

> Pair with a friend once via a ticket or QR; after that, click their card to drop a file.

WAN by default. End-to-end encrypted. No accounts, no krill-operated infrastructure.

Part of the [krill](https://krill.software) umbrella of small, calm, single-purpose Linux apps. See [SPEC.md](SPEC.md) for the full design.

## Status

Pre-v1. Currently at M1: first-run card setup. The networking layer (iroh) is not wired in yet.
